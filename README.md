# DevForge AI Backend

A Spring Boot REST API for managing software projects.

## Features

* CRUD Operations
* PostgreSQL Database
* Spring Data JPA
* DTO Pattern
* Validation
* Global Exception Handling
* Swagger/OpenAPI Documentation

## Tech Stack

* Java 21
* Spring Boot 3.5
* PostgreSQL
* Maven

## API Endpoints

* GET /api/projects
* GET /api/projects/{id}
* POST /api/projects
* PUT /api/projects/{id}
* DELETE /api/projects/{id}

## Run Locally

```bash
./mvnw spring-boot:run
```

Swagger UI:

```
http://localhost:8081/swagger-ui.html
```
